.clamp(num, low, up)

It returns the number in the middle of the other two numbers. Note that bottom should always be lower than top.
(他の2つの数値の中間に位置する数値を返します。 bottomにtopより大きい数は指定できません。）

.inRange(num, start, end)

It returns if num is in the middle of start and end. If you set a number into end which is lower than start, they will be exchanged.
numがstartとendの間にあるかどうかを調べます。もしstartにendよりも大きい数字をセットした場合2つの値は入れ替えて計算されます。
